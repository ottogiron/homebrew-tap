# Homebrew Tap

Homebrew formulae for [ottogiron](https://github.com/ottogiron) projects.

## Available Formulae

| Formula | Description |
|---------|-------------|
| [compas](https://github.com/ottogiron/compas) | Multi-agent orchestrator for AI-assisted development |

## Install

```bash
brew install ottogiron/tap/compas
```

## Upgrade

```bash
brew upgrade ottogiron/tap/compas
```

## Uninstall

```bash
brew uninstall compas
brew untap ottogiron/tap  # optional — removes the tap itself
```

## Troubleshooting

If a formula fails to install, try:

```bash
brew update
brew install ottogiron/tap/compas
```

For issues with a specific formula, file a bug on the formula's source repository (linked in the table above).
